# CLTD M1 User Test Bundle

This folder is the shareable web version of the Module 1 interactive study guide.

Files:
- `index.html`: the app-like study experience
- `manifest.webmanifest`: install metadata for Add to Home Screen behavior

Best way to share with a friend:
1. Upload this folder to any static host.
2. Send them the hosted URL.
3. Have them open it in Safari on iPhone.
4. If desired, they can use `Share` -> `Add to Home Screen`.

Easy hosting options:
- GitHub Pages
- Netlify Drop
- Vercel static site
- Cloudflare Pages

Important:
- Sharing the raw `.html` file through Files/iCloud usually opens Preview instead of Safari.
- A normal HTTPS web link is the right setup for user testing.
